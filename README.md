## Overview
This repository is python package for setup of NetWork simulation using Cognac; MD simulator.

このリポジトリは、統合シミュレータである OCTA における MD シミュレータ Cognac を用いて、ネットワークポリマーのシミュレーションを行うための UDF を作成するパイソンパッケージです。
